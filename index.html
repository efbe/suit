suit is a two-layered, modular UI library.

Structure apart from design

While the structure part resets, sets up and positions the elements or controls core units like the gutter width; the theme layer handles the design related CSS properties like block heights, colors or border sizes. This way you can easily redesign elements by keeping the basic rules in order and reduce the divergence among UI units. The main reason behind the design layer nested into structure and not laying on the top of it is to avoid rules getting overridden. The motto of suit is Extend, not overwrite.

Blasted into modules

Modularity in suit means each and every component has its own compiled CSS. Building from small standalone bricks like a single button to complex apps like an entire styleguide suit allows you to only include what your page needs. However you can compile all of the selected elements into one file as well which is available throughout your entire web page if you are more of a cache fan.
