Default grid

suit has a 12 column grid system which is extendable with a gutter module.

In mobile view all columns have full width and the gutter size becomes the bottom margin of the blocks.














Grids with gutters

Add .gutter class to the .grid-row for proper whitespace between columns. The gutter size comes from the _global-variables.scss.













Nesting grids

Inserting grids without gutter into each other is quite straightforward.




Nested grids with gutter will always inherit the whitespace between columns...




...so if you want to keep the initial gutters even when you are nesting consider using mixed .grid-row-s until you get what you want.




You can also nest layouts without gutters into columns with gaps using the .no-gutter class.

Nesting works for three levels, if you need more then you should think your layout over.




Stick-to-left

The gutter module has a small stick-to-left extension which might come handy if you are lazy and don't want to nest the shit out the grids, but in need of a menu column which sticks onto the left side of the viewport and does not mess up the other columns and gaps.

No more than two of this panel can be used within one grid.
