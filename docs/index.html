Suit is a two-layered, modular UI library.

While the structure part resets, sets up and positions the elements or controls core units like the gutter width. The theme layer handles the design related CSS properties like block heights, colors or border sizes. This way you can easily redesign elements by keeping the basic rules in order and reduce the divergence among UI units.

The layers are in separated SASS files collected in the structure and theme folders. Each structure file has a matching theme file and includes it with the SASS @import rule. There's a Grunt task which is responsible for merging these two layers into one CSS file. For instance dog.scss that contains the structural part and imports _dog.scss with the design layer compiles into dog.css.

The unit variables are stored in the _global-variables.scss which is equally used by both layers. The colors are stored in the _colors.scss that is only accessible from the design layer. Therefor you can easily redesign all of your UI elements through these two config files.
