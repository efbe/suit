Suit is a two-layered, modular UI library where the structure part resets, sets up and positions the elements while the theme part handles the design related CSS rules from gutters to block heights and from colors to border sizes declared in a config file thus you can create custom themed elements by keeping the core properties and reduce the divergence among UI units.

The layers are in separated SASS files collected in the structure and theme folders. Each structure file has a matching theme file and includes it with the SASS @import rule. There's a Grunt task which is responsible for merging these two layers into one CSS file.
