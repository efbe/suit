Default grid

suit has a 12 column grid system which is extendable with a gutter module.

In mobile view all columns have full width and the gutter size becomes the bottom margin of the blocks.















Grids with gutters

Add .gutter class to the .grid-row for proper whitespace between columns. The gutter size comes from the _global-variables.scss.














Nesting grids

Grids with gutter inherit the proper whitespace between columns.



You can also nest layouts without gutters into columns with whitespace using the .no-gutter class.




Sticky grids

The gutter module also has a small .stick-to-[side] feature which might come handy if you want a side menu which sticks onto the side of the viewport but does not mess up layout columns with gutters.
